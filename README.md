# Nebula Thrift — AI-powered Cloud Cost Optimizer

> Stop the bleed of inefficient cloud spending. Nebula Thrift scans, analyzes, and optimizes your cloud infrastructure in real-time.

## Overview

Nebula Thrift connects to your AWS account (Azure & GCP coming soon), identifies wasted resources, and generates AI-powered recommendations with quantified savings.

## Tech Stack

- **Backend:** Python, FastAPI, AWS Lambda, boto3
- **AI Engine:** Claude API (Anthropic), LangChain
- **Frontend:** React, Vite, TailwindCSS, Recharts
- **Infrastructure:** Terraform, GitHub Actions CI/CD
- **Cloud:** AWS (S3, DynamoDB, Lambda, EventBridge, CloudFront, Cognito)
