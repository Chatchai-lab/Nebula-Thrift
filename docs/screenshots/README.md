- **Breakpoint:** Desktop 1440px (oder mit Suffix `-tablet`, `-mobile`)

- **Ohne Browser-Chrome:** nur die Seite, kein Adressleiste/Bookmarks
- **Naming:** `<seite>-<variante>.png`, z.B. `landing-hero.png`, `dashboard-mobile.png`
